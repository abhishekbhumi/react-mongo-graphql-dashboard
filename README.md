# react-mongo-graphql-dashboard
React Mongo GraphQL Dashboard is a full-stack web application that combines a React frontend with a MongoDB + GraphQL backend, deployed across Firebase Hosting and Render. It features a secure login system, device and IP tracking, and a scalable comment and bookmark management


Tech Stack
- Frontend: React + Vite + Apollo Client
- Backend: Express + Apollo Server + MongoDB
- Auth: JWT-based login with context-aware tracking
- Deployment: Firebase Hosting (frontend) + Render (backend)
- Extras: CORS handling, environment-based endpoint switching, user-agent parsing
  
Features
- Login flow with IP and device fingerprinting
- Comment editing with “edited” status tracking
- Bookmark grouping and filtering
- Admin-ready backend schema with scalable models
- Dynamic environment switching for local and production
- Clean CORS configuration for cross-origin access
